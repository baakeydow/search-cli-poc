# Custom-Search-Index
POC search words in files

## Prerequisites
[Java 11.0.5](https://www.eclipse.org/openj9)  
[Maven 3.5.4](https://maven.apache.org/download.cgi)  

## Run Project
>```mvn clean package && java -jar target/doftom-search-cli.jar .```  
--- or ---  
>```sh run_docker_container.sh```
